## 🎯 프로젝트 소개

이 프로젝트는 **KDT12 Python 과정**에서 학습한 내용을 종합적으로 활용한 미니 프로젝트입니다.

사용자가 건강 정보(나이, 키, 몸무게, 혈압 등)를 입력하면 다음을 분석합니다:

- **BMI (체질량지수)** 계산 및 비만도 판정
- **혈압 상태** 분석 (정상 ~ 2기 고혈압)
- **심혈관 질환 위험도** 평가 (Kaggle 데이터 기반)
- **맞춤형 건강 조언** 제공

---

## ✨ 주요 기능

### 1. BMI 분석
- WHO 아시아-태평양 기준 적용
- 저체중 / 정상 / 과체중 / 비만 / 고도비만 판정

### 2. 혈압 분석
- 대한고혈압학회 기준 적용
- 저혈압 ~ 2기 고혈압 6단계 분류

### 3. 심혈관 질환 위험도
- 다중 위험 요소 기반 점수 산출
- 나이, BMI, 혈압, 콜레스테롤, 혈당, 생활습관 종합 평가

### 4. 건강 기록 관리
- CSV 파일로 기록 저장
- 기록 조회 및 히스토리 관리

### 5. Kaggle 통계
- 70,000건 심혈관 데이터 기반 평균값 비교
- 샘플 데이터 통계 확인

## 📖 사용 방법

### Step 1: 기본 정보 입력
- 이름, 나이, 성별, 키, 몸무게 입력

### Step 2: 건강 지표 입력
- 수축기/이완기 혈압 (mmHg)
- 콜레스테롤 수치 선택
- 혈당 수치 선택
- 생활습관 체크 (흡연, 음주, 신체활동)

### Step 3: 분석하기
- [분석하기] 버튼 클릭
- BMI, 혈압, 위험도 결과 확인
- 맞춤 건강 조언 확인

### Step 4: 기록 관리
- [기록 저장]: 현재 분석 결과 저장
- [기록 조회]: 저장된 기록 확인
- [통계 보기]: Kaggle 데이터 통계 확인

## 📊 데이터 출처

**Kaggle - Cardiovascular Disease Dataset**
- URL: https://www.kaggle.com/datasets/sulianova/cardiovascular-disease-dataset
- 데이터 수: 70,000건
- 특성: 나이, 성별, 키, 몸무게, 혈압, 콜레스테롤, 혈당, 흡연, 음주, 신체활동, 심혈관질환 여부

### 데이터 컬럼

| 컬럼 | 설명 |
|------|------|
| age | 나이 (일수 → 년 변환) |
| gender | 성별 (1: 여성, 2: 남성) |
| height | 키 (cm) |
| weight | 몸무게 (kg) |
| ap_hi | 수축기 혈압 (mmHg) |
| ap_lo | 이완기 혈압 (mmHg) |
| cholesterol | 콜레스테롤 (1: 정상, 2: 높음, 3: 매우 높음) |
| gluc | 혈당 (1: 정상, 2: 높음, 3: 매우 높음) |
| smoke | 흡연 (0: 비흡연, 1: 흡연) |
| alco | 음주 (0: 비음주, 1: 음주) |
| active | 신체활동 (0: 비활동, 1: 활동) |
| cardio | 심혈관질환 (0: 없음, 1: 있음) |

## 📌 주요 판정 기준

### BMI (WHO 아시아-태평양)

| BMI | 판정 |
|-----|------|
| < 18.5 | 저체중 |
| 18.5 ~ 22.9 | 정상 |
| 23.0 ~ 24.9 | 과체중 |
| 25.0 ~ 29.9 | 비만 |
| ≥ 30.0 | 고도비만 |

### 혈압 (대한고혈압학회)

| 분류 | 수축기 | 이완기 |
|------|--------|--------|
| 저혈압 | < 90 | < 60 |
| 정상 | < 120 | < 80 |
| 주의 | 120~129 | < 80 |
| 고혈압 전단계 | 130~139 | 80~89 |
| 1기 고혈압 | 140~159 | 90~99 |
| 2기 고혈압 | ≥ 160 | ≥ 100 |
